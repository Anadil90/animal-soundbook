# animal-soundbook

## animal soundbook is a simple web app that plays animal sounds when the user (usually a child) clicks or taps on an animal image tile. The app was designed with the intention to make learning animal names and animal sounds more enjoyable and fun. The app will also serve as an experiment to gauge the interest of children in learning.
